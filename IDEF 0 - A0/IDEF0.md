# A-0 — Gerenciar Operações do Hospital Universitário

| Tipo        | Elementos                                                                 |
|-------------|----------------------------------------------------------------------------|
| **Entrada** | Dados de pacientes; Solicitações de atendimento; Pedidos de exames; Encaminhamentos externos |
| **Controle**| Leis e normas de saúde; Políticas do hospital; Protocolos clínicos; Contratos de convênios; Regras da universidade |
| **Mecanismo** | Profissionais de saúde; Equipe administrativa; Sistema de gestão hospitalar; Infraestrutura; Banco de dados |
| **Saída**     | Atendimentos realizados; Exames e procedimentos concluídos; Registros clínicos e históricos; Faturamento gerado; Relatórios gerenciais; Dados para ensino e pesquisa |



# A1 — Gerenciar Cadastro de Pacientes e Profissionais

| Tipo        | Elementos                                                                 |
|-------------|----------------------------------------------------------------------------|
| **Entrada** | Dados de pacientes; Dados de profissionais; Documentos pessoais (CPF, CRM, COREN) |
| **Controle**| Políticas de cadastro; Normas de privacidade (LGPD); Regras de credenciamento |
| **Mecanismo** | Sistema de cadastro; Equipe administrativa; Banco de dados |
| **Saída**     | Pacientes cadastrados; Profissionais cadastrados; Planos de saúde registrados |



# A2 — Planejar e Agendar Consultas, Exames e Leitos

| Tipo        | Elementos                                                                 |
|-------------|----------------------------------------------------------------------------|
| **Entrada** | Solicitações de atendimento; Disponibilidade de profissionais; Disponibilidade de leitos/salas |
| **Controle**| Regras de prioridade; Protocolos de agendamento; Horários e restrições dos setores |
| **Mecanismo** | Sistema de agendamento; Equipe de recepção; Agenda dos profissionais |
| **Saída**     | Consultas e Exames agendadas; Procedimentos marcados; Leitos reservados; Filas de atendimento |



# A3 — Executar Atendimentos e Registrar Informações Clínicas

| Tipo        | Elementos                                                                 |
|-------------|----------------------------------------------------------------------------|
| **Entrada** | Agenda de atendimentos; Prontuário do paciente; Resultados de exames; Dados de cadastro |
| **Controle**| Protocolos clínicos; Normas de segurança; Regras de acesso ao prontuário |
| **Mecanismo** | Médicos; Enfermeiros; Técnicos; Equipamentos médicos; Sistema de prontuário eletrônico |
| **Saída**     | Diagnósticos; Procedimentos realizados; Solicitações de exames; Evoluções; Alta hospitalar; Histórico atualizado |



# A4 — Gerenciar Faturamento e Financeiro

| Tipo        | Elementos                                                                 |
|-------------|----------------------------------------------------------------------------|
| **Entrada** | Procedimentos realizados; Dados de convênios e planos; Tabelas de preços |
| **Controle**| Regras de faturamento; Contratos com convênios; Normas fiscais |

| **Saída**     | Contas geradas; Guias de convênio; Relatórios financeiros; Demonstrativos de cobrança |



# A5 — Gerar Relatórios, Auditoria e Apoio a Ensino/Pesquisa

| Tipo        | Elementos                                                                 |
|-------------|----------------------------------------------------------------------------|
| **Entrada** | Dados de atendimentos; Dados financeiros; Histórico médico; Logs de auditoria |
| **Controle**| Políticas de auditoria; Requisitos de pesquisa; Regras de acesso a dados; Exigências legais |

| **Saída**     | Relatórios gerenciais; Relatórios de auditoria; Indicadores; Dados anonimizados para pesquisa |
